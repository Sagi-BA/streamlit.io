# אתר אישי - שגיא בר און

זהו פרויקט אתר אישי עבור שגיא בר און, המציג סדנאות והרצאות בתחום הבינה המלאכותית.

## דרישות מקדימות

כדי להריץ את האתר, יש צורך בהתקנת הדרישות הבאות:

- Python 3.9 ומעלה
- pip (מנהל החבילות של Python)

## התקנה

1. יש להוריד את כל קבצי הפרויקט למחשב שלך.

2. ליצור תיקייה בשם `assets` בתוך תיקיית הפרויקט.

3. להוסיף לתיקיית `assets` את הקבצים הבאים:
   - `profile.jpg` - תמונת פרופיל
   - `balloons.json` - קובץ אנימציית Lottie (אופציונלי - אם אין קובץ, האפליקציה תשתמש בקובץ מקוון)

4. להתקין את כל החבילות הנדרשות על ידי הרצת הפקודה הבאה בטרמינל/cmd:

```
pip install -r requirements.txt
```

## הפעלת האתר

לאחר התקנת כל החבילות הנדרשות, ניתן להריץ את האתר באמצעות הפקודה:

```
streamlit run app.py
```

האתר יפתח אוטומטית בדפדפן בכתובת המקומית (בדרך כלל `http://localhost:8501`).

## מבנה הפרויקט

- `app.py` - הקובץ הראשי של האפליקציה
- `requirements.txt` - רשימת החבילות הנדרשות לפרויקט
- `assets/` - תיקייה המכילה נכסים כמו תמונות וקבצי אנימציה
  - `profile.jpg` - תמונת הפרופיל שתוצג באתר
  - `balloons.json` - קובץ אנימציית Lottie (אופציונלי)

## התאמה אישית

ניתן להתאים את האתר באמצעות עריכת הקובץ `app.py`:

1. שינוי פרטי הסדנאות וההרצאות
2. עדכון המלצות לקוחות
3. שינוי אירועים קרובים
4. התאמת העיצוב באמצעות שינוי ה-CSS בתחילת הקובץ

## אופציה לקבלת קובץ Lottie

אם ברצונך ליצור קובץ Lottie משלך, ניתן להשתמש באתרים הבאים:
1. LottieFiles: https://lottiefiles.com/
2. Icons8: https://icons8.com/animated-icons

לאחר הורדת קובץ JSON של האנימציה, יש לשמור אותו בתיקיית `assets` בשם `balloons.json`.

## פריסה לאינטרנט

ניתן לפרוס את האתר באמצעות שירותים כמו:
1. Streamlit Sharing: https://streamlit.io/sharing
2. Heroku: https://heroku.com
3. Render: https://render.com

לפריסה לאינטרנט, יש ליצור קובץ `Procfile` עם התוכן:
```
web: streamlit run app.py
```

## תמיכה

אם יש לך שאלות או בעיות, אנא צור קשר דרך האתר לאחר הפעלתו.
